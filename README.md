# Data Volley to SQL
A program used to convert Data Volley Files (.dvw) into a SQL database

## Dependencies
- openvolley
  - https://github.com/openvolley/datavolley
- R packages:
  - assertthat
  - data.table
  - enc
  - jpeg
  - lubridate
  - plyr
  - dplyr
  - raster
  - readr
  - sp
  - gt
  - dbplyr
  - RSQLite
  - rio
